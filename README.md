# Hotel-Dioniso


## 1. Descripción del proyecto

Hotel Dionisio es una **aplicación web de gestión de reservas de hotel** que permite a los usuarios:

- Consultar información de las habitaciones y servicios del hotel  
- Reservar y cancelar habitaciones  
- Gestionar su perfil de usuario  
- Acceder a información general del hotel (instalaciones, ubicación, contacto)

El proyecto combina **React**, **Supabase** y **Tailwind CSS** para ofrecer una experiencia moderna, intuitiva y segura.

---

## 2. Tecnologías utilizadas

- **React**: Librería para construir interfaces de usuario dinámicas y reactivas  
- **Tailwind CSS**: Framework de utilidades para estilos rápidos y responsivos  
- **Supabase**: Backend como servicio, incluyendo:
  - Base de datos PostgreSQL
  - Autenticación y autorización
  - API REST para gestión de reservas  

---

## 3. Funcionalidades principales

- Registro y autenticación de usuarios  
- Visualización de habitaciones disponibles y filtrado por fechas  
- Realización y cancelación de reservas  
- Gestión y edición del perfil de usuario  
- Control de acceso según sesión iniciada
